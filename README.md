# 基于时间序列分析的降雨量预测系统 python1106

## 项目概述

本项目是一个基于时间序列分析的降雨量预测系统，采用Python技术栈，结合B/S结构以及MYSQL数据库实现。系统旨在通过数据分析，为用户提供准确可靠的降雨量预测功能。

## 技术栈

- **后端语言**: Python
- **前端结构**: B/S（浏览器/服务器）
- **数据库**: MYSQL

## 功能模块

### 用户管理

- 用户登录：用户需要输入用户名和密码进行身份验证。
  - 用户名和密码验证
- 用户管理：
  - 用户列表查看
  - 用户增加与删除

### 降雨量预测

- 输入预测年份和月份
- 点击“开始预测”按钮
- 系统在几秒钟内处理并显示预测数据

## 系统角色

- **普通用户**：可以登录系统，进行降雨量预测。
- **管理员**：负责用户管理，包括用户的增加与删除。

## 运行环境

- Python环境
- MYSQL数据库服务器
- 支持现代浏览器的操作系统

## 部署步骤

1. 安装Python运行环境
2. 安装并配置MYSQL数据库
3. 部署后端服务
4. 部署前端页面
5. 启动服务
